# Clarity Aluminum Website Starter

Static website starter for Clarity Aluminum Windows & Doors LLC.

## What is included
- `index.html` homepage
- `projects/haggins-residence.html` starter project page
- `assets/styles.css` responsive styling
- `assets/script.js` small mobile menu script

## Recommended free hosting
Best simple option: GitHub Pages.

## Deploy to GitHub Pages
1. Create a GitHub account.
2. Create a new repository, for example `clarity-aluminum-website`.
3. Upload all files in this folder.
4. Go to Settings > Pages.
5. Under Build and deployment, choose Deploy from a branch.
6. Select `main` branch and `/root` folder.
7. Save. GitHub will publish the site.

## Optional later upgrades
- Buy a domain, for example `clarityaluminum.com`.
- Replace placeholder photos with real project photography.
- Add downloadable PDF catalog, NFRC certificates, and warranty documents.
- Replace mailto form with Netlify Forms, Formspree, or a custom backend.
- Add Google Analytics and Search Console.
